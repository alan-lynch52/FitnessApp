Fitness App API

Format: JSON

GET /users/:username/:password

Get a user’s details

GET /users/:username/:password/:recordList

Get a record list of a given type for a given user

GET /users/:username/:password/:recordList/:date

Get a record list of a given type, all records from a given date

GET /users/:username/:password/:recordList/exID/:exID

Get a record list of a given type, all records with a given exID
